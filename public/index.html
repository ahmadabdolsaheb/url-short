URL-shortener usage Example:

Entering the following url

https://urlsh8.herokuapp.com/new/https://www.youtube.com/

will return a json with a shortened url number 

{
"original": "https://www.youtube.com/",
"shortened": "6237986",
"_id": 6237986
}

if the shortened url is used such as the following:

https://urlsh8.herokuapp.com/6237986

the app will redirect you to the original website
